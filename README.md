## 一、介绍

**(这里不需要自己另外添加环境，标准的python环境即可) **

**主要功能：** 可以根据单个md文件地址或者md目录文件的地址，批量提取md里面图片的地址，并且把md图片对应的地址修改成提取图片对应的位置。

**格式：** 会在md文件同目录下建立一个images文件夹，用来存放系统转移过来的文件。

## 二、md目录统一提取图片到images文件夹中

```she
# 在py文件所在文件夹，打开cmd
python md目录图片移动.py
```
